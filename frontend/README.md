# Agri World Frontend

Next.js App Router frontend for the Agri World FastAPI backend.

## Setup

```bash
cd frontend
copy .env.example .env.local
npm install
npm run dev
```

App URL: http://localhost:3000

## Environment

- `NEXT_PUBLIC_API_BASE_URL=http://localhost:8000`

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run type-check`
